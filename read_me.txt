This is a modified AXEL program with limited capabilities for SIMPLE usage:
- Can provide download link
- Can specify number of connections
- Can specify maximum download speed

How to use:
-------------
>> Double click "run_this.bat" file
>> Provide the parameters as asked
>> File is downloaded to the "output" directory
